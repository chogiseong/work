2. setupbuild.sh 관련 업무 공유

안녕하십니까.
iMotors IVI 개발 3팀 조기성입니다.

기존에는 브랜치를 전부 맞게 맞추고 ap_build.sh 스크립트를 통해 환경설정 후 빌드를 하는 방식이었습니다.
하지만 최근에 모비스에서 setupbuild.sh 스크립트를 사용하는 경우가 많아 업무 공유 드립니다.

커맨드 예시

source setupbuild.sh -d CE1 MASTER master 0.31.0 master

source커맨드 / setupbuild.sh 스크립트 선택 / 빌드 옵션(1) / 차종 / 형태(master, P2, P1 등) / meta-mobis 브랜치 / meta-sonatus 브랜치 / meta-autoever 브랜치 / + 기타 추가 옵션 (없어도 정상적으로 빌드 가능합니다.)

(1)
-h   show list of command-line options
-v   skipping Sonatus version check
-e   skipping ESW FW version check
-d   enable debug image
-s   enable SSH

예시로 CE1 Master를 빌드하려면

source setupbuild.sh -d CE1 MASTER master 0.31.0 master 이런식의 커맨드를 사용하시면 됩니다.
sonatus 브랜치는 옵션으 스킵할수도 있고, 제대로 입력되지 않으면 0.31.0 등을 사용하라고 에러 메세지로 나오기 때문에 거기에 맞게 빌드 하시면 됩니다.
